# Commands


## Docker-এর সব Command ব্যাখ্যা

---

### 🐳 IMAGE সম্পর্কিত Commands

| Command | কাজ |
|---|---|
| `docker build .` | current directory থেকে image build করো |
| `docker build -t myapp:1.0 .` | নাম ও tag দিয়ে image build করো |
| `docker images` | সব local image দেখো |
| `docker pull ubuntu` | Docker Hub থেকে image download করো |
| `docker push myapp` | Docker Hub-এ image upload করো |
| `docker rmi myapp` | image delete করো |
| `docker tag myapp myapp:v2` | image-এ নতুন tag দাও |
| `docker inspect myapp` | image-এর বিস্তারিত তথ্য দেখো |
| `docker history myapp` | image-এর layer history দেখো |
| `docker save myapp > myapp.tar` | image কে file-এ save করো |
| `docker load < myapp.tar` | file থেকে image load করো |

---

### 📦 CONTAINER সম্পর্কিত Commands

| Command | কাজ |
|---|---|
| `docker run ubuntu` | image থেকে container চালাও |
| `docker run -it ubuntu bash` | interactive terminal সহ চালাও |
| `docker run -d myapp` | background-এ (detached) চালাও |
| `docker run -p 8080:80 myapp` | port mapping করো (host:container) |
| `docker run -v /data:/app myapp` | volume mount করো |
| `docker run --name mycon myapp` | container-এ নাম দাও |
| `docker run --rm myapp` | শেষ হলে container auto-delete |
| `docker run -e KEY=VALUE myapp` | environment variable দাও |
| `docker ps` | চলমান containers দেখো |
| `docker ps -a` | সব containers দেখো (বন্ধটাও) |
| `docker stop mycon` | container বন্ধ করো (gracefully) |
| `docker kill mycon` | container জোর করে বন্ধ করো |
| `docker start mycon` | বন্ধ container আবার চালু করো |
| `docker restart mycon` | container restart করো |
| `docker rm mycon` | container delete করো |
| `docker rm -f mycon` | চলমান container জোর করে delete |
| `docker exec -it mycon bash` | চলমান container-এ ঢুকো |
| `docker logs mycon` | container-এর log দেখো |
| `docker logs -f mycon` | log real-time follow করো |
| `docker inspect mycon` | container-এর বিস্তারিত দেখো |
| `docker stats` | সব container-এর CPU/RAM usage দেখো |
| `docker top mycon` | container-এর running process দেখো |
| `docker cp file.txt mycon:/app` | file container-এ copy করো |
| `docker cp mycon:/app/file.txt .` | container থেকে file বের করো |
| `docker rename old new` | container-এর নাম পরিবর্তন করো |
| `docker pause mycon` | container pause করো |
| `docker unpause mycon` | container unpause করো |
| `docker wait mycon` | container বন্ধ হওয়া পর্যন্ত অপেক্ষা করো |
| `docker diff mycon` | container-এ কী পরিবর্তন হয়েছে দেখো |
| `docker commit mycon myimage` | container থেকে নতুন image বানাও |

### 🌐 NETWORK সম্পর্কিত Commands

| Command | কাজ |
|---|---|
| `docker network ls` | সব network দেখো |
| `docker network create mynet` | নতুন network তৈরি করো |
| `docker network rm mynet` | network delete করো |
| `docker network inspect mynet` | network-এর বিস্তারিত দেখো |
| `docker network connect mynet mycon` | container-কে network-এ যুক্ত করো |
| `docker network disconnect mynet mycon` | network থেকে container বিচ্ছিন্ন করো |
| `docker run --network mynet myapp` | নির্দিষ্ট network-এ container চালাও |

### 💾 VOLUME সম্পর্কিত Commands

| Command | কাজ |
|---|---|
| `docker volume ls` | সব volume দেখো |
| `docker volume create myvol` | নতুন volume তৈরি করো |
| `docker volume rm myvol` | volume delete করো |
| `docker volume inspect myvol` | volume-এর বিস্তারিত দেখো |
| `docker volume prune` | অব্যবহৃত সব volume মুছো |

---

### 🧹 CLEANUP / SYSTEM Commands

| Command | কাজ |
|---|---|
| `docker system df` | disk usage দেখো |
| `docker system prune` | অপ্রয়োজনীয় সব কিছু মুছো |
| `docker system prune -a` | সব unused image ও container মুছো |
| `docker container prune` | বন্ধ সব container মুছো |
| `docker image prune` | dangling image মুছো |
| `docker image prune -a` | unused সব image মুছো |

---

### 🔐 REGISTRY / LOGIN Commands

| Command | কাজ |
|---|---|
| `docker login` | Docker Hub-এ login করো |
| `docker logout` | Docker Hub থেকে logout করো |
| `docker search ubuntu` | Docker Hub-এ image খোঁজো |

### 🚀 DOCKER COMPOSE Commands

| Command | কাজ |
|---|---|
| `docker compose up` | সব service চালু করো |
| `docker compose up -d` | background-এ চালু করো |
| `docker compose down` | সব service বন্ধ করো |
| `docker compose build` | সব image build করো |
| `docker compose ps` | compose container দেখো |
| `docker compose logs` | সব service-এর log দেখো |
| `docker compose exec app bash` | নির্দিষ্ট service-এ ঢুকো |
| `docker compose restart` | সব service restart করো |
| `docker compose stop` | সব service stop করো |
| `docker compose pull` | সব image pull করো |

### ℹ️ INFO Commands

| Command | কাজ |
|---|---|
| `docker version` | Docker-এর version দেখো |
| `docker info` | Docker system-এর সব তথ্য দেখো |
| `docker help` | সাহায্য দেখো |

---

### 🧠 সহজে মনে রাখার উপায়:

```
image    → build, pull, push, rmi, images
container → run, ps, stop, start, rm, exec, logs
network  → network ls/create/rm/connect
volume   → volume ls/create/rm
cleanup  → system prune
```


## Create docker network

docker network create mongo-network 

## start mongodb

docker run -d \
-p 27017:27017 \
-e MONGO_INITDB_ROOT_USERNAME=admin \
-e MONGO_INITDB_ROOT_PASSWORD=password \
--name mongodb \
--net mongo-network \
mongo

## start mongo-express
docker run -d \
-p 8085:8081 \
-e ME_CONFIG_MONGODB_ADMINUSERNAME=admin \
-e ME_CONFIG_MONGODB_ADMINPASSWORD=password \
--net mongo-network \
--name mongo-express \
-e ME_CONFIG_MONGODB_SERVER=mongodb \
mongo-express   

## open mongo-express from browser

http://localhost:8081
